Karten-Bank

Ziehe Karten in einen Slot von 0 bis 15 oder zurück in die Bank.

Format: Eine Liste aus Objekten mit name und optional image. Wenn kein Bild angegeben ist, wird automatisch ein Platzhalter mit Initialen erzeugt.

Mittel-Slot ist fest auf 8.
Die ausgewählte Starterkarte wird beim Start automatisch auf die Mitte gesetzt und gesperrt.
Beispiel-Thema: Streamer mit den meisten Followern.
Du bist Gamemaster und ziehst die Karten je nach Antwort deiner Freunde nach links oder rechts.
Für GitHub Pages reicht diese eine Datei als index.html.
